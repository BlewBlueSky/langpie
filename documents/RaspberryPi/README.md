# Raspberry Pi 树莓派软件准备

## 操作系统镜像

* 强烈推荐: raspbian 
```bash
Distributor ID:	Raspbian
Description:	Raspbian GNU/Linux 10 (buster)
Release:	10
Codename:	buster
```
* 镜像连接: www.raspberrypi.org/downloads/

## 烧录工具

* Etcher: etcher.io
* Win32_diskimager: 请自行百度

## 远程登录工具

* Putty
* Xshell
* SecureCRT
* Rshell
* Powershell

## 文件传输工具

* Windows: 推荐 filezilla / sftp 
* Mac: scp
* Linux: scp
